# Tic-Tac-Toe
Этот проект я создал для проверки своих знаний после довольно большого перерыва и оценки своих сил при последующих проектах.
Основная цель данного проекта — вспомнить основные механизмы и структуру работы с Unity.

## Игровой процесс
Я думал изначально создать обычную игру «Крестики-Нолики», но после некоторых размышлений я понял, что хочу сделать необычную игру, в которой будет свой мрачный стиль и интересные фишки.
